You can use this application to check the information about each building in the municipality area.

To see the detailed information about each building you have to:

  • Zoom to the building location. The buildings are showed at the scale 1:2k or greater (1:2k, 1:1k, 1:500 or 1:200)
  • Select one or more buildings, by clicking them. The selected buildings are added to the Selected buildings table.
  • Select the building on the Selected buildings table.
